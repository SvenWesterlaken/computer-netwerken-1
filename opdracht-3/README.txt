Omdat de onderwerpen van de emails gedecodeerd zijn in UTF-8 is
het nodig om in de console ook UTF-8 aan te zetten indien dit is
meegekomen in de zip. Zo worden een aantal karakters niet getoond 
met een vraagteken.

Aanwezige library is voor het decoderen van Quoted-Printable decoding.

Dit kan bij IntelliJ via:
Settings > Editor > File Encodings > Project Encoding
